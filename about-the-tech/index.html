About the Tech

De technische keuzes achter de gratis tools. Kort, concreet en zonder marketingtaal.

Architectuur

Volledig frontend: pure HTML, CSS en JavaScript zonder backend afhankelijkheden.

State management via localStorage zodat tools ook offline blijven werken.

Geen externe APIs; alleen optionele CDN's voor PDF-export zodat de basis licht blijft.

Waarom frontend-only

Minder attack surface en geen backend om te beheren of te patchen.

Snellere laadtijden door minimale payloads en geen auth flows.

Gebruikers houden controle: alle berekeningen draaien lokaal zonder data-exfiltratie.

Privacy-first

Geen tracking pixels, geen analytics en geen accounts.

Persoonlijke gegevens verlaten de browser niet; lokale opslag kan je zelf resetten.
