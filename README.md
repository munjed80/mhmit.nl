# MHM IT — Company Website and Product Hub

This repository hosts the new static website for MHM IT (mhmit.nl).
The goal is to build a fast, secure, fully responsive business website using:
• HTML5 + CSS3 + JavaScript (no WordPress, no PHP)
• Mobile-first design
• Premium dark metallic theme with electric blue and purple accents
• Modern UI components and icons

The website must promote MHM IT services and products.

## Pages to Implement
1. Home (Landing Page)
   - Hero section: short tagline + CTA button
   - MHM IT value proposition
   - Primary services/cards
   - Featured products/tools
   - Short portfolio preview

2. Services
   - Details of service categories:
     Web development, Shopify optimization, SEO, Security/Monitoring
     SaaS & automation solutions
     Custom dashboards and integrations

3. Products / Tools
   Showcase the following:
   - Smart ZZP Hub
   - ChefSense
   - Product Ad Generator
   - SmartOrderHub for restaurants
   - MHM UBA Automations
   - Custom business dashboards

4. About
   - Company mission
   - Expertise (IT, automation, SaaS)
   - Short founder bio

5. Contact
   - Contact form (Email API to be integrated later)
   - WhatsApp CTA
   - Map or location reference (optional)

## Technical Requirements
• Fast performance (high Lighthouse score)
• Clean semantic code with accessibility in mind
• Reusable CSS components and variables
• Sticky header + smooth scrolling
• Separate CSS and JS files
• All assets in `assets/` folder
• Setup for GitHub Pages deployment
